Database Foundations: Administration

Ongoing, regular administration is critical to the security and performance of databases such as Oracle and SQL Server. It's also a key topic of the Microsoft Technology Associate (MTA) Exam. Whether you're studying to pass the test or simply to keep your admin skills up to date, this course will cover the most current techniques and best practices for administering a database. Adam Wilbert covers the core concepts, including securing the server with user authentication and roles, assigning object-level permissions, and performing a backup and restore.

Learning Objectives:

-Securing the database server

-Understanding Windows authentication vs. SQL Server authentication

-Assigning fixed server roles and fixed database roles

-Granting object-level permissions

-Understanding ownership claims

-Creating backups

-Restoring a database
